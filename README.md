# war card game v2

web-based version of the "War" card game.<br>
this is an improved version of a previous project.

## built with

- svelte
- sveltekit
- javascript
- tailwindcss
- lucide icons

## installation

- `git clone https://github.com/drae528/war-card-game-v2.git`
- `cd war-card-game-v2`
- `npm install`
- `npm run dev`
